# ZohoCrm

Read data in Zoho CRM by using API.

## Installation

Add this line to your application's Gemfile:

    gem 'zoho_crm'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install zoho_crm

## Usage

    require "zoho_crm"
